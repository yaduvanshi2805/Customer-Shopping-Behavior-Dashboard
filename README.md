🛍️ Customer Shopping Behavior Dashboard

# 📌 Project Overview

Customer Shopping Behavior Dashboard is an End-to-End Data Analytics Project developed using Python, SQL, and Power BI.

The project demonstrates how raw customer shopping data can be cleaned, transformed, analyzed, and visualized to generate meaningful business insights.

It follows a real-world analytics workflow that is commonly used by Data Analysts to support business decision-making.


# 🎯 Project Goal

The primary goal of this project is to transform raw customer shopping data into meaningful business insights that help organizations understand customer behavior and improve decision-making.

The project focuses on:

- Understanding customer purchasing patterns
- Identifying high-performing product categories
- Measuring customer spending behavior
- Analyzing subscription trends
- Evaluating customer reviews and ratings
- Building an interactive dashboard for business users
- Demonstrating a complete End-to-End Data Analytics workflow


# 💼 Business Problem

Retail companies collect large volumes of customer transaction data every day. However, raw data alone cannot support strategic decision-making.

This project converts raw customer data into interactive reports that help answer important business questions such as:

- Which product categories generate the highest revenue?
- Which customer segment spends the most?
- How do subscriptions influence purchasing behavior?
- Which age groups contribute the highest sales?
- What are the overall customer satisfaction trends?

- Clothing generated the highest revenue.
- Young Adult customers contributed the highest sales.
- Majority of customers are non-subscribers.

- Average Review Rating is **3.75**.
- Customer purchasing behavior varies across different categories and age groups.
